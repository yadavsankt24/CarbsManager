We're listening and we appreciate your feedback! This article includes answers to some of your questions.

Tip: New to Carb Manager? See Get started with Carb Manager.

Why did you update the app?

It's been several years since Carb Manager had a significant update, and we were overdue for one. We're always collecting feature requests and feedback from customers, and many of the updates in this app redesign (version 7) are based on that feedback, including:

  • Multi-add: This much-requested feature means you can now add foods directly from the food finder without having to open a food detail dialog. Just swipe right on the food, or tap the Multi-Add button, then tap the + button to add your most recently-selected serving size.
  • Smart Search: Use "natural language" in your search queries, like "two eggs and a cup of coffee with cream" to bring up all three items at once.
  • Macro Goals for each meal: Premium members can now set specific macro goals for each meal via Settings > Meal Goals.
  • New features for people with diabetes, including HbA1c, eAG tracking, and the ability to specify insulin types.
  • Custom water "glass" size for quicker tracking.
  • Better accessibility & zoom functionality: Increase text size under  Settings >   App Settings > Additional Settings.
  • New Keto and low carb learning opportunities in the extensive Keto Academy video series.

We’ve spent the past 18 months reimagining and redesigning the Carb Manager app from the ground up, with a new look and many new features that make it even easier — and more efficient — to manage your diet and fitness goals all in one place.

How do I sync data between the mobile app and web app in version 7?

  • Use the same email address and password for your registered account when you sign in to version 7 of the mobile app and version 7 of the web app at my.carbmanager.com. When you are signed in to the same account, your data will automatically sync between the mobile app and web app.

Why can’t I see some of the recent changes in the new app?

For the best performance, ensure you get the latest updates to Carb Manager version 7. You may need to refresh the mobile app or web app as described in the following sections.

Refresh the mobile app

If the mobile app is open, even in the background, when we release an update, you must fully close the app before the app can be refreshed with the latest version.

  • Force Carb Manager to close on your device, then reopen Carb Manager to automatically get the latest version.
    Tips to force an app to close:

    iOS devices: Follow the steps on Apple's help center.

    Android devices:

    1. Open the Settings app.
    2. Scroll down and select Apps or Application Manager.
    3. Select Carb Manager.
    4. Tap Force Stop to fully close the app.
  • If you suspect that you still don't see the latest updates in the mobile app, uninstall the app and then reinstall the app from the App Store or Google Play.
    Note: If you don't have a registered account, we recommend that you create one before you uninstall the mobile app. Otherwise, you will lose access to your logs and other data when you uninstall the app.

Refresh the web app

The web app, my.carbmanager.com, always opens in the most current version, but if we release an update while you are using the app you will see a message asking you to refresh the app.

  • If a green Refresh message appears on the screen, click Refresh to get the latest update.

How do I scan barcodes or QR codes?

You can log foods by scanning UPC or EAN barcodes from food labels. You can also scan QR codes from our Carb Manager's Keto Diet Cookbook.

  • Select Add Item >  Scan Food. (Use the same button to snap pictures.)

For more information, see Log foods by scanning barcodes and snapping pictures.

How do I find My Foods, Recipes, and Favorites?

You can now find custom foods, recipes, meals, meal plans, and favorites in the main food search window.

  • Select Add Item >  Add Food, then select the My Foods tab.

For more information, see Find foods, recipes, and meals, which includes a tutorial video.

How do I update my weight, measurements, and other health metrics daily?

  • Select  Goals in the main navigation.

My Measurements is now in Goals. You can add your measurements, see your history, and also update your goals for measurements.

Tip: If you don't see your whole history, check the time period that is selected.

For more information, see View progress and gain insights toward goals, which includes a tutorial video.

How do I change the text size?

  • Go to  Settings >  App Settings > Additional Settings and select Text Size.

Where are the forums?

  • To access the Carb Manager low carb & Keto diet forums, go to forums.carbmanager.com.
Tip: You can also access message-style discussion groups from within Carb Manager. For more information, see Groups in Carb Manager.

Where is the In-Depth tab?

You can now view your daily, "in-depth" details straight off the Daily Log home screen.

  • Go to the  Daily Log, then select the In-Depth Details button. (On a small screen, the button appears at the end of the list of daily log entries. On a large screen, the button appears on the right side of the screen.)

For more information, see View progress charts of daily nutrition goals and totals > In-depth details.

How do I import a recipe?

Premium members still have the option to import recipes from certain websites.

  • Select  Add Item >  Add Food >  Options > Import Recipe.

For more information, see Import recipes.

How do I create a custom food, recipe, or meal?

  • Select  Add Item >  Add Food >  Options and select either Create Food, Create Recipe, or if you are a Premium member, Create Meal, or Create Meal Plan.

For more information, see the following articles.

  • Create custom foods.
  • Create custom recipes, which includes a tutorial video.
  • Create custom meals.
  • Create custom meal plans.

How do I edit a custom food, including editing serving units and weights?

  1. Select  Add Item >  Add Food.
  2. Select the My Foods tab.
  3. Select the custom food that you want to edit.
  4. In the custom food's screen, select  Options > Edit Food, then update details about the food.

Here is a short video that shows you how to change the serving weight and the serving unit.

Tip: While editing a custom food, you can also edit the name, folder, and other details about the custom food. For more information, see Edit custom foods.
Tip: Want to edit a food that is in a custom recipe? Because recipes are made up of individual foods, first copy the food that you want to edit to My Foods, then edit the food as described in the previous steps.
